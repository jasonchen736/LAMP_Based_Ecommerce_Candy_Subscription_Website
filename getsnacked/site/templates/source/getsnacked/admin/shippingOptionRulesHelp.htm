Shipping Option Rules are options that allow you to specify when and where a shipping option will be available.

To do this, you must first set the rule type. This will describe how all rules for the option are handled.
The rule types are:

  • None:  Rules will be ignored and the option will be available to all orders
  • Allow:  Only available to orders matching the given rules, the shipping option will not be available to any other orders
  • Block:  The shipping option will be available to all orders except for the orders that match the given rules

After a rule type is set, you may provide any number of rules that you choose.
The rule fields are as follows:


Weight, package and cost rules must have a condtion. The conditions are:

  • [=]  Equal to
  • [>]  Greater than
  • [>=]  Greater than or equal to
  • [<]  Less than
  • [<=]  Less than or equal to

Provinces may be spelled out, while state values must be 2 character abbreviations.
Fields are not case sensitive. Any field left blank will be ignored and will not factor in to the match.

Example: An allow rule set with a zip code value of 10002 and weight >= 5 will only be available for orders in the zip code 10002 with a weight greater than or equal to 5 lbs.
